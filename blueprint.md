# **Project Blueprint: MaracatuTech Landing Page**

## **1. Overview**

This document outlines the design, features, and implementation plan for the MaracatuTech landing page. The goal is to create a visually stunning, modern, and informative single-page website for a fictional tech event that merges technology with the vibrant culture of Maracatu.

## **2. Design & Style Guide**

- **Aesthetic:** A bold, energetic, and premium feel that blends cultural vibrancy with a dark, tech-focused theme.
- **Color Palette:**
  - **Primary Background:** A very dark grey (`#121212`) with a subtle noise texture.
  - **Accent Colors:** `oklch(70% 0.25 290)` (vivid magenta), `oklch(80% 0.2 145)` (bright cyan-green).
  - **Text:** Off-white (`#EAEAEA`).
- **Typography:**
  - **Headlines:** 'Orbitron', sans-serif.
  - **Body:** 'Roboto', sans-serif.
- **Visual Effects:**
  - **Drop Shadows:** Multi-layered, soft drop shadows for depth.
  - **Glow Effect:** Subtle, colored `box-shadow` for interactive elements.
- **Layout:** Responsive, adapting from a single column on mobile to a grid on larger screens.

## **3. Features & Sections Implemented**

- **Header & Navigation:** Sticky header with logo and smooth-scrolling navigation.
- **Hero Section:** Large headline, tagline, and a prominent "Register Now" CTA button.
- **About Section:** A two-column section with descriptive text and an image, explaining the event's vision.
- **Speakers Section:** A responsive grid of speaker cards with placeholder images and titles, featuring a "lifted" shadow effect on hover.
- **Footer:** Contains social media links and copyright information.
